# GA4GH-DEMO
This repo contains DEMO RAG Implementation on first 3 pages of GA4GH Framework for Responsible Sharing Document.
## Tech Stack
- LangChain – RAG pipeline orchestration
- Pinecone – Vector database (Hybrid search)
- BM25 Encoder – Sparse retrieval
- HuggingFace Embeddings – Dense vector representations
- Groq API (llama-3.1-8b-instant) – LLM inference
- Streamlit – Frontend UI
